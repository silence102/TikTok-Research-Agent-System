# Tutorial Generator Skill

## 목적

비기술자도 따라할 수 있는 단계별 튜토리얼을 자동 생성합니다.

## 튜토리얼 구조

### 1. 목표 설정
```markdown
## 튜토리얼 목표

이 튜토리얼을 완료하면:
✅ [구체적 성과 1]
✅ [구체적 성과 2]
✅ [구체적 성과 3]

소요 시간: [예상 시간]
난이도: 초급/중급/고급
```

### 2. 사전 준비
```markdown
## 준비물

### 필수
- [ ] [도구 1] (버전 X.X 이상)
- [ ] [도구 2]
- [ ] [계정/API 키]

### 선택
- [ ] [옵션 1]
```

### 3. 단계별 가이드

각 단계는 다음 형식을 따릅니다:

```markdown
### Step X: [단계 제목] (소요: X분)

**목표**: [이 단계에서 달성할 것]

**작업**:
1. [세부 작업 1]
   ```bash
   # 실행 명령어
   ```

2. [세부 작업 2]
   - [체크포인트 1]
   - [체크포인트 2]

**확인**:
- [ ] [검증 항목 1]
- [ ] [검증 항목 2]

**문제 해결**:
| 문제 | 원인 | 해결 |
|------|------|------|
| [오류 메시지] | [원인] | [해결 방법] |
```

## 작성 원칙

### 명확성
- 전문 용어 사용 시 항상 설명 추가
- 스크린샷 또는 코드 블록으로 시각화
- "여기를 클릭" 대신 "상단 메뉴의 Settings 클릭"

### 검증 가능성
- 각 단계마다 체크리스트 제공
- 예상 출력 결과 명시
- 문제 해결 섹션 포함

### 점진적 복잡도
- 간단한 것부터 시작
- 각 단계는 이전 단계 성공을 전제
- 선택적 고급 기능은 별도 섹션
