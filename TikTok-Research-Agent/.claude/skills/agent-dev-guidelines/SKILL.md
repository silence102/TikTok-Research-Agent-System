# Agent Development Guidelines Skill

## 핵심 원칙

이 스킬은 삼양식품 DXT팀의 바이브코딩 방법론을 기반으로 에이전트를 개발합니다.

### 1. 맥도날드 시스템 철학

```
본사 (테크리드) → CLAUDE.md 200줄 가이드 작성
지점장 (바이브코더) → 의도와 의사결정 기록
알바 (AI 에이전트) → 매뉴얼대로 실행
```

### 2. 7단계 워크플로우

1. **컨텍스트 확보**: 요구사항 파악
2. **요구사항 정의**: PRD 작성 (Goal/Non-Goal/Must-have)
3. **UX 설계**: 사용자 여정 정의
4. **의존성 분석**: 기술 스택 검토
5. **구현 계획**: 단계별 작업 분해
6. **리스크 분석**: 예상 문제점 파악
7. **테스트 작성**: 검증 시나리오 준비

### 3. 3대 원칙

- **빠른 시작**: CLAUDE.md 200줄 이내로 핵심만
- **어펜딕스 참조**: 상세 내용은 도메인별 문서 분리
- **휘발성 컨텍스트**: 중요한 것만 Git 커밋

## 사용 방법

에이전트 개발 시:
- PRD 먼저 작성
- CLAUDE.md는 200줄 엄수
- docs/ 폴더에 도메인별 분리 (Ideation, PRD, Tasks, TechStack, Tutorial)
