# 삼양식품 DXT팀 프롬프트 스타일 가이드

## 핵심 원칙

### 1. 비즈니스 임팩트 중심
프롬프트 작성 시 "왜 이 정보가 필요한가?"를 명시합니다.

**Bad**:
```
"틱톡 트렌드를 알려줘"
```

**Good**:
```
"Popow 플랫폼에서 브랜드에게 추천할 틱톡 트렌드 Top 5를 분석해줘.
각 트렌드가 마이크로 인플루언서(1K-50K)에게 적합한 이유를 포함해."
```

### 2. 실험 우선 문화 반영

프롬프트에 "먼저 작동하게, 그다음 개선"을 녹여냅니다.

**초기 프롬프트** (빠른 검증):
```
"주어진 데이터에서 해시태그 상위 10개를 추출하세요."
```

**개선 프롬프트** (정확도 향상):
```
"주어진 데이터에서 해시태그 상위 10개를 추출하되,
1. 스팸성 해시태그 제외 (예: #followme, #likeforlike)
2. 브랜드 마케팅 관련성 높은 것 우선
3. 각 해시태그의 조회수 변화율 포함"
```

### 3. 출처 투명성

모든 답변에 데이터 출처를 명시하여 신뢰성을 확보합니다.

**시스템 프롬프트 예시**:
```
답변 형식:

**핵심 정보**: [요약]
**상세 내용**: [설명]
**데이터 출처**:
- [소스 1]: [URL/날짜]
- [소스 2]: [URL/날짜]
```

## 도메인별 프롬프트 패턴

### 틱톡 트렌드 분석
```
역할: 틱톡 마케팅 전문가
목표: Popow 플랫폼의 브랜드-크리에이터 매칭 최적화
데이터: Airtable tiktok_trends 테이블
지침:
1. 마이크로/나노 인플루언서 관점에서 분석
2. 브랜드 협업 가능성 평가
3. 실행 가능한 인사이트 제공
```

### 크리에이터 이코노미 리서치
```
역할: 크리에이터 이코노미 리서처
목표: 최신 업계 동향을 주간 리포트 형태로 요약
데이터: Airtable research_news 테이블
지침:
1. Popow 비즈니스와 관련성 높은 뉴스 우선
2. 경쟁사 동향 포함
3. 향후 3개월 예측 포함
```

## Temperature 설정 가이드

| 작업 유형 | Temperature | 이유 |
|----------|-------------|------|
| 데이터 추출 | 0.0 | 정확성 최우선 |
| 요약 | 0.3 | 일관성 + 약간의 표현 다양성 |
| 인사이트 생성 | 0.7 | 창의적 해석 필요 |
| 트렌드 예측 | 0.5 | 균형 잡힌 추론 |
