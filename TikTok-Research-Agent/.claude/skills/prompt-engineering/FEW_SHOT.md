# Few-Shot Learning Examples

## 틱톡 트렌드 분석

### Example 1: 해시태그 분석
**Input**:
```
질문: "지난주 가장 인기 있었던 해시태그는?"
데이터: [{"hashtag": "#fyp", "views": "12M", "date": "2025-01-13"}, ...]
```

**Output**:
```
지난주(2025-01-06~01-13) 가장 인기 있었던 해시태그는 #fyp로 조회수 12M을 기록했습니다.
(출처: Apify TikTok Scraper, 수집일: 2025-01-13)
```

### Example 2: 데이터 없음 처리
**Input**:
```
질문: "AI 관련 틱톡 트렌드는?"
데이터: []
```

**Output**:
```
현재 데이터베이스에 AI 관련 틱톡 트렌드 정보가 없습니다.
데이터 수집 범위를 확인하거나 검색 키워드를 변경해주세요.
```

## 크리에이터 이코노미 분석

### Example 3: 뉴스 요약
**Input**:
```
원본: "TikTok announces new creator fund program..."
```

**Output**:
```
**제목**: 틱톡, 크리에이터 펀드 프로그램 신규 발표
**요약**: 마이크로 인플루언서(1K-50K 팔로워) 대상 수익화 지원
**임팩트**: Popow 타겟 크리에이터 직접 지원 → 경쟁력 강화 필요
(출처: TechCrunch, 2025-01-10)
```

## RAG 컨텍스트 구성

### Example 4: 다중 소스 종합
**Input**:
```
질문: "최근 1주일 틱톡 마케팅 트렌드는?"
데이터: [해시태그 10개] + [뉴스 5개]
```

**Output**:
```
**해시태그 트렌드**: #smallbusiness, #brandpartnership 급상승
**업계 뉴스**: 브랜드-마이크로 인플루언서 협업 증가 추세
**인사이트**: 소규모 브랜드가 나노 크리에이터와 협업하는 사례 증가 중

출처:
- Apify 해시태그 데이터 (2025-01-13)
- Social Media Today (2025-01-11)
```
